# go-build-configurator
création d'une légère surcouche de `go build` pour ajouter un système de configuration par fichier json ou yaml


# Installation
Télécharger l'executable :
- Rendez vous sur la [liste des Gitiob Actions](https://github.com/nicolachoquet06250/go-build-configurator/actions)
- Cliquez sur le dernier Github Action en date
- Téléchargez l'artefact correspondant à votre système

Installer le binaire :
- Dézipez l'archive
- Ouvrez
  - Pour windows
    - Powershell
  - Pour linux & mac 
    - un terminal
- Lancez la commande suivante :
  - `cd go-build-configurator-<os>-<arch>`
  - Pour windows :
    - `.\go-build-configurator.exe --install`
  - Pour linux & mac :
    - `./go-build-configurator --install`
  - Si vous êtes sur windows, fermez et re ouvrez Powershell pour que les modifications soient pris en compte
